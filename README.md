# To-Do List App

Aplikasi web pencatatan tugas sederhana untuk UAS Web Dasar.

## Fitur
- Tambah & hapus tugas
- Tandai tugas selesai
- Dark / Light mode
- Data disimpan menggunakan file text (PHP)

## Teknologi
- PHP
- HTML
- CSS
- JavaScript

## Catatan
Aplikasi dijalankan menggunakan local server (XAMPP).
